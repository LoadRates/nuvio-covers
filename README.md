# Nuvio Covers

Animated focus covers (GIFs) for Nuvio collections — used as visual headers for genres, studios, streaming services, decades, and award categories.

## Contents

The `gifs/` folder contains 84 animated GIF covers.

## Usage

Files are served via jsDelivr CDN:

```
https://cdn.jsdelivr.net/gh/LoadRates/nuvio-covers@main/gifs/<name>.gif
```
